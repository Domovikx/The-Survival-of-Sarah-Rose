# Плагины (Plugins)

Плагины расширяют OpenCode кастомными инструментами, хуками и интеграциями.

**Документация:** https://opencode.ai/docs/plugins/

**Установка:**
1. Через npm: добавь в `opencode.json` → `plugin: ["package-name"]`
2. Локально: помести файл плагина в `.opencode/plugins/`
