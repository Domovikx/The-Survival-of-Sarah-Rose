## Назначение

Этот скил запускает параллельную обработку нескольких файлов с помощью `task(general, ...)`.
Каждый файл получает отдельного `general`-агента со своей изолированной сессией.

## Входные данные

Список путей к .rpy файлам (относительно корня проекта).

## Процесс

1. Для КАЖДОГО файла из списка запусти ОДНИМ СООБЩЕНИЕМ (одной пачкой):
   task(general, "Переведи файл {file} на русский.

   Инструкция:
   1. Загрузи skill-translate-rpy через skill({{ name: \"skill-translate-rpy\" }})
   2. Переведи все строки где new == old
   3. Загрузи skill-review-rpy через skill({{ name: \"skill-review-rpy\" }})
   4. Проверь качество перевода
   5. Если ревью rejected — исправь замечания и повторяй translate→review снова (до 10 раз)
   6. Если approved — верни \"OK {file} (попыток: N)\"
   7. Если после 10 попыток не прошёл — верни \"FAIL {file}: <причина>\"
   8. ВАЖНО: НЕ изменяй old строки, только new
   ")

2. Дождись результатов от всех task(general)
3. Верни итоговый отчёт

## Ограничения

- Не более 5 параллельных task(general) одновременно
- Каждый general работает независимо в своей сессии

## Формат отчёта

```
ГОТОВО: X/Y файлов переведено (Z с ошибками)
```

Список ошибок:
- file.rpy — <причина>
