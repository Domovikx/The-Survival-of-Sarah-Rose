## When to use me

Проверяю качество перевода:

- "проверь качество этого перевода"
- "найди ошибки в этой строке"
- "проверь файл на ошибки"
- "найди ляпы"

## Проверки

### 1. Поиск пустых переводов

```bash
grep -rn '    ""' tl/<lang>/script/split/
```

### 2. Поиск дубликатов блоков

```bash
grep -n 'translate <lang> SceneName_hash' file.rpy | sort | uniq -d
```

### 3. Проверка на слова исходного языка в переводе

Ищу типичные артефакты: слова исходного языка, вкраплённые в перевод.

### 4. Проверка логических ошибок перевода

- Перевод полностью не соответствует оригиналу
- Ошибки согласования (род, число, падеж)
- Грубые смысловые ошибки

### 5. Проверка орфографии

- Двойные буквы
- Пропущенные буквы
- Очевидные опечатки

### 6. Проверка формата

```rpy
translate <lang> unique_id:

    # "оригинал"
    "перевод"
```

## Критерии качества

✅ Перевод качественный когда:
- Нет пустых переводов
- Нет дубликатов блоков
- Нет слов исходного языка в переводе
- Нет грубых логических ошибок
- Правильный формат
- Хорошее согласование (род, число, падеж)

⚠️ Перевод требует исправления когда:
- Есть пустые строки
- Есть дубликаты блоков
- Есть слова исходного языка
- Есть грубые логические ошибки
- Неправильный формат
- Ошибки согласования
